이미지 관련 태그

>

src 속성으로 이미지의 경로를 설정하여 화면에 이미지를 표시할 수 있다.
절대 경로 :
C:/, D:/ 와 같은 보조 기억장치 시작부터
해당 파일이 위치한 경로를 모두 서술하는 것
ex) D:/수업자료/workspace/HTML5/resources/image/river1.PNG
상대 경로 :
현재 파일을 기준으로 다른 파일의 위치를 찾아가는 것
./ or 파일명으로 시작 : 현재 파일과 같은 폴더 안의 위치에서 시작한다.
../ : 현재 파일이 위치한 폴더 바깥으로 나가서 찾는다.
ex)
'0_firstPage.html' : 같은 폴더 안에 있음.
'../index.html' : 현재 폴더를 나가서 찾겠다!
'../../../img.jpg' : 3번 나가서 img 이미지를 찾아라!

alt 속성

사진의 경로가 잘못되었거나, 이미지가 훼손되었을 경우, 스크린 리더기(화면 낭독기)를 통해 사진을 표현하거나, 글자로 사진의 정보를 나타내려는 용도로 사용하는 속성이다.

width 와 height

고정 길이 : 화면 크기가 변해도 사진은 그대로

가변 길이 : 화면 크기에 따라 사진도 크기가 변하는 설정
